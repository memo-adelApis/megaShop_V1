# 🛒 MegaShop – Full-Stack E-commerce Platform + Admin Panel

MegaShop is a complete **Full-Stack E-commerce platform** built with **Next.js** and **MongoDB**, offering a modern shopping experience for users along with a powerful Admin Panel for managing store operations.

## 🚀 Features

### 🛍 Storefront (User Side)
- Product browsing with categories & brands
- Product details page
- Add to cart, edit cart, remove items
- Checkout process (billing + order placement)
- User authentication & authorization with next-auth
- Order tracking
- Responsive UI built with Tailwind CSS

### ⚙️ Admin Panel
- Manage Products (CRUD)
- Manage Categories & Brands
- Manage Users
- Manage Orders & order status
- Dashboard with statistics and insights
- Secure admin routes

### 🔐 Authentication
- next-auth (Credentials Provider)
- Secure Sessions & JWT
- Role-based Access Control (Admin/User)

### 🗄 Backend & Database
- Next.js API Routes
- MongoDB + Mongoose models
- Clean, modular folder structure (components, ui, modules...)

### 🛠 Tech Stack
- **Next.js (App Router)**
- **React**
- **Tailwind CSS**
- **MongoDB**
- **next-auth**
- **Node.js**
- **Context API**
